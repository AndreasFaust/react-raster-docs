Getting Started

PeerDependencies#

  • react: >= 16.8.0,
  • react-dom: >= 16.8.0,
  • styled-components: >= 5.2.0

Install#

Install all dependencies with your favourite package-manager:

  • Yarn
  • npm
yarn add react-raster styled-components react react-dom
npm install --save react-raster styled-components react react-dom

Usage#

react-raster has only two components: Grid and Box.

  • Grid is the wrapper, that defines the gridâ€™s basics like gutter, breakpoints and other things.
  • Box is the element, that gets positioned inside the Grid.

Boxes and Grids can be freely nested inside each other.

import {Box, Grid} from "react-raster";
Live Editor
function BasicExample() {
  return (
    <Grid
      colspan={[6,6,12]}
      left={"2vw"}
      right={"2vw"}
      top={"2vw"}
      bottom={"2vw"}
      gutterX={"2vw"}
      gutterY={"2vw"}
      control
    >
        <Box 
          cols={[3,3,6]}
          css={`background: blue;`}
          alignX="center"
          alignY="center"
          height="200px"
        >
            <h2>Hello</h2>
        </Box>
        <Box 
          cols={[3, 3, 6]} 
          css={`background: yellow;`}
          alignX="center"
          alignY="center"
        >
          <h2>World!</h2>
        </Box>
    </Grid>
  );
}
