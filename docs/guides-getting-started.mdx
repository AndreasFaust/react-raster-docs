### Basic Usage

- **react-raster** has only one component called `Box`.
- Define `brekpoints` and a `colspan` to start a new Grid and nest `Box`-Elements inside each other, preserving the Grid.
- Further style your `Box`-Elements directly via **props**.
- Boxes and Grids can be freely nested inside each other.
- If you set `control`, you can press `Esc` to see the grid.

```js
import Box from "react-raster";
```

```jsx live
function BasicExample() {
  return (
    <Box
      breakpoints={[0, 400, 800, 1200]}
      colspan={6}
      padding={"1rem"}
      gridRowGap={"1rem"}
      gridColumnGap={"1rem"}
      control
    >
      <Box
        as="h1"
        margin="0"
        cols={[6, 6, 3]}
        padding="2rem"
        height={"500px"}
        background="lightblue"
        display="flex"
        alignItems="center"
        justifyContent="center"
      >
        Hello World!
      </Box>
      <Box cols={[6, 6, 3]} height="500px">
        <Box
          as="p"
          margin="0"
          display="flex"
          color="white"
          alignItems="center"
          justifyContent="center"
          background="darkblue"
          fontSize="1.5rem"
          fontWeight="bold"
        >
          Stop
        </Box>
        <Box
          as="p"
          margin="0"
          display="flex"
          color="white"
          alignItems="center"
          justifyContent="center"
          background="coral"
          fontSize="1.5rem"
          fontWeight="bold"
        >
          Wars
        </Box>
      </Box>
    </Box>
  );
}
```
