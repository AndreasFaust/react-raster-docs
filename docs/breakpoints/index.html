Breakpoints

Defining Breakpoints#

Breakpoints is an array of numbers starting with 0. Every number defines a min-width.

Default Breakpoints#

[0, 432, 768, 1024, 1200, 1400]

Props matching Breakpoints#

Often you want to bind props to certain breakpoints. You can achieve this by defining an array instead of a single string or number for your prop.
